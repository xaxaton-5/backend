## Что нужно

- Docker
- Docker Compose

**1. Клонировать репозиторий**

```
git clone https://github.com/xaxaton-5/backend
```

**2. Перейти в папку с проектом**

```
cd backend
```

**3. Запустить контейнер**

```bash
docker compose up --build
```
